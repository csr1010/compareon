# Compareon - AI Product Comparison

Compare products across the web and make smarter buying decisions with AI-powered insights.

## What is Compareon?

Compareon helps you compare products from across the web intelligently. Add products to your comparison list as you browse, then get AI-powered analysis to help you choose the best option for your needs.

**No login required. No personal information needed. Just smart shopping.**

## Key Features

- **Compare Across the Web**: Add products from different sites to one comparison
- **AI-Powered Insights**: Get intelligent analysis and recommendations
- **No Account Needed**: Start comparing immediately - no registration required
- **Privacy First**: No personal data collected, no login, completely anonymous
- **Smart Organization**: View all your products with images, prices, and ratings
- **Auto-Sync**: Your comparison list is saved automatically

## How to Use

### Adding Products
1. Browse online stores and search for products
2. Click the "Compare" button that appears on product pages
3. Products are automatically added to your comparison list

### Comparing Products
1. Click the Compareon icon in your browser toolbar
2. Review your added products with all their details
3. Click "Start Comparing" to get AI-powered insights
4. Make smarter buying decisions with comprehensive analysis

### Managing Your List
- Remove products by clicking "Remove" under the product image
- Add up to 5 products for comparison
- Use the refresh button (↻) to sync your list

## Privacy & Security

- **No login or registration required** - use immediately
- **No personal information collected** - completely anonymous
- **No tracking or profiling** - your privacy is protected
- All communications are encrypted (HTTPS)
- Your comparison data is only used to provide insights

## Installation

### Option 1: Chrome Web Store (Recommended - Coming Soon)
1. Visit the Chrome Web Store
2. Click "Add to Chrome"
3. Start browsing and adding products to compare

### Option 2: Install from Source (Available Now)

**Method A: Clone from Git**
1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/compareon.git
   cd compareon
   ```

2. Open Chrome and navigate to `chrome://extensions/`

3. Enable **Developer mode** (toggle in the top-right corner)

4. Click **"Load unpacked"**

5. Select the `compareon` folder you just cloned

6. The extension is now installed! Look for the Compareon icon in your browser toolbar

**Method B: Download ZIP**
1. Download the repository as a ZIP file from GitHub
   - Click the green "Code" button
   - Select "Download ZIP"

2. Extract the ZIP file to a folder on your computer

3. Open Chrome and navigate to `chrome://extensions/`

4. Enable **Developer mode** (toggle in the top-right corner)

5. Click **"Load unpacked"**

6. Select the extracted folder

7. The extension is now installed! Look for the Compareon icon in your browser toolbar

### Updating the Extension
If you installed from source and want to update to the latest version:
1. Pull the latest changes: `git pull` (for Git method)
2. Or download and extract the latest ZIP (for ZIP method)
3. Go to `chrome://extensions/`
4. Click the refresh icon (↻) on the Compareon extension card
